基本操作

1.概要

Sapiens Poker Loggerは、ポーカーの遠征記録・セッション管理・ハンド履歴をテキスト形式で記録するPWAアプリです。ブラウザだけで動作し、オフラインでも使えます。

Journey作成 旅のタイトル
→
セッション開始 場所・ブラインド
→
ハンド記録 カード・アクション
→
セッション終了 キャッシュアウト
データ保存: 全データはブラウザのlocalStorageに保存されます。アカウント登録は不要です。ブラウザのデータ削除や端末の故障で記録が失われる場合があるため、CSV ExportやHands Exportで定期的にバックアップを取ることをおすすめします。

2.トップページ

アプリ起動時に表示される画面です。作成したJourneyがアコーディオン形式で一覧表示されます。

Journeyカード

  • 各Journeyのタイトル・日付・統計情報を表示
  • タップで展開し、セッション一覧がインラインで表示される（画面遷移なし）
  • 展開時にはJourney内の統計（Total P/L / Sessions / Total Hours / BB/hr）も確認可能

ガイドボックス

  • Journeyが存在しない場合は展開状態で表示。基本的な使い方と出力サンプルが表示される
  • Journeyが存在する場合は折りたたまれ、タップで展開可能

フッター

  • Start New Journey — 新しい遠征・旅の記録を作成
  • Manual — このマニュアルを別タブで開く
  • Feedback — フィードバックフォームを開く
  • X アイコン — X（旧Twitter）へのリンク
  • カラーパレットアイコン — カラーカスタマイズ画面を開く（セクション22参照）

3.Journey・セッション・ハンドの階層管理

Journey・セッション・ハンドはすべてトップページのアコーディオンで階層的に管理されます。

Journeyの展開

  • Journeyカードをタップすると展開され、セッション一覧が表示される
  • Journey内の統計情報（Total P/L / Sessions / Total Hours / BB/hr）を確認可能

セッションの展開

  • セッション項目をタップすると展開され、詳細統計・ハンド一覧が表示される
  • 各セッションには日付・場所・ゲーム情報・ハンド数・P/Lが表示される
  • 展開したセッション内にハンド一覧・「Add Hand」「Export」ボタンが表示される

Journey内のボタン

  • New Session — 展開したJourney内に表示。新しいセッションを開始
  • Edit — Journey右上に表示。タップでJourney編集モーダルが開く（タイトル・日付の変更、削除が可能）
  • CSV Export — 完了セッションがある場合に表示

セッション

4.セッション管理

セッション開始

  1. Journey内で「New Session」をタップ
  2. Location（場所）、Game Type、SB/BB、Buy-in を入力
  3. 「Start」でセッション開始。経過時間の計測が始まります

Game Type（ゲーム種別）

以下の5種類から選択できます:

  • NLH — No Limit Hold'em（デフォルト）
  • PLO — Pot Limit Omaha
  • PLO5 — Pot Limit Omaha 5-card
  • Limit — Fixed Limit
  • Mixed — Mixed Game

セッション中の操作

ボタン機能
AddセッションバーのBuy-in表示横のボタン。追加バイインを記録。複数回可能
Endキャッシュアウト額を入力してセッション終了。収支が自動計算されます
Handsハンド一覧パネルの表示/非表示を切り替え。バッジに保存済みハンド数が表示される

セッションバー

セッション中は画面上部にセッション情報バーが表示されます。場所・ブラインド・経過時間・Buy-in合計が確認できます。

BB/hr計算: ブラインドが途中で変わった場合、各フェーズの経過時間で加重平均BBを算出し、正確なBB/hrを計算します。

5.セッションの編集

セッションの編集方法は2つあります。

1. Editモーダル（セッション展開時）

セッション展開内の「Edit」ボタンをタップすると編集モーダルが開きます。

項目操作
Locationテキスト入力で変更
Start/End 日時日付ピッカーと時・分の数値入力（ホイール/スワイプ対応）で編集
Game Typeドロップダウンで変更（NLH / PLO / PLO5 / Limit / Mixed）
SB / BB数値入力で変更

モーダル下部の「Delete This Session」ボタンでセッション削除も可能です。

2. インライン編集（セッション詳細画面）

セッション詳細画面では、破線アンダーラインの付いた項目をタップして直接編集できます。

項目操作
Locationタップ → テキスト入力に切替。Enter または フォーカスアウトで保存
Dateタップ → 日時編集モーダルが開く
Gameタップ → テキスト入力に切替
Buy-insタップ → 各Buy-in金額を個別に編集できるモーダルが開く
Cash-outタップ → 数値入力に切替

P/L・Duration・BB/hr・Hands は自動計算のため直接編集はできません。

セッションを再開

詳細画面の「セッションを再開」ボタンで、終了したセッションをアクティブ状態に戻せます。経過時間の計測も再開され、ハンドの追加記録やAdd Onが可能になります。

6.編集・削除の操作

Edit ボタンの配置

  • Journey — 展開したJourney内の右上「Edit」ボタン → 編集モーダル（タイトル・日付変更、削除）
  • セッション — セッション展開内の右上「Edit」ボタン → 編集モーダル（Location・日時・Game Type・SB/BB変更、削除）
  • ハンド — 各ハンド項目の「Edit」ボタン → ハンド編集画面が開く

削除

  • Journey削除 — Journey編集モーダル内の「Delete This Journey」ボタン
  • セッション削除 — セッション編集モーダル内の「Delete This Session」ボタン、またはセッション項目の「DEL」ボタン
  • セッション詳細画面 — 画面下部の「Delete Session」ボタン
注意: 削除は確認ダイアログが表示されます。削除後のデータ復元はできません。

ハンド記録

8.ハンド記録の流れ
  1. Hand / Blind セクションでハンド番号・ブラインド・Hero情報を入力
  2. Cards セクションでHeroカード・ボードカードを選択
  3. Players セクションでプレイヤーの着席状況を設定
  4. 「テンプレート生成」ボタンでフルスクリーンエディタが開く
  5. アクションを追記して「Save」で保存
情報入力 Hand / Blind / Cards
→
テンプレート生成 自動フォーマット
→
アクション追記 エディタで編集
→
Save セッションに保存

Hand / Blind セクション

項目説明
Hand #ハンド番号（1〜99）。Save後に自動インクリメント
SB / BBブラインド金額
Anteアンティ金額（0で省略）
Blind Presetよく使うブラインドを一覧から選択（15種類: 1/2, 1/3, 2/3, 2/5, 5/5, 10/20, 25/50, 50/100, 100/200, 200/300, 200/400, 200/500, 300/500, 400/800, 500/1000）
Hero NameHeroの名前
StackHeroのスタック金額
SAVEBlind / Hero / Stack のデフォルト値を保存

Cards セクション

  • Hero Cards — 2枚のカードスロット。タップでカードピッカーが開く
  • Board Cards — Flop 3枚 + Turn 1枚 + River 1枚の計5スロット
  • 入力済みスロットの「×」ボタンでカードを除去
  • スロットは任意の順序で入力可能

Players セクション

  • 各シート（1〜9）にプレイヤー名を入力
  • D ボタン — タップでそのシートをBTN（ディーラー）に設定
  • シート番号 — タップでそのシートをHeroに設定（反転表示になる）
  • × ボタン — シートの有効/無効を切替（無効シートはテンプレートから除外）
  • ポジション名（UTG, MP, CO, BTN, SB, BB等）は人数に応じて自動割り当て
  • SAVE — デフォルト人数を保存
補足: ショーダウンやリザルト等の項目は、本ツールが元々ポーカーのプレイシーン映像を構築するためのデータ入力ツールとして設計されているためです。ハンドログとしてのみ使用する場合は省略可能です。

カスタムキーボード & 入力支援

18.カスタムキーボード

フルスクリーンエディタでは、ポーカー専用のカスタムキーボードが表示されます。3つのモードを切り替えて効率的に入力できます。

Action Mode（デフォルト）

ハンドのアクションを素早く入力するためのモードです。

キー種別キー
数字0〜9
記号# /
アクションx(check) c(call) b(bet) r(raise) f(fold) ai(allin) win chop
ナビゲーション← →（カーソル移動）
編集␣(スペース) ⌫(バックスペース) ↵(改行)
モード切替⇄ CARD（カードモードへ） ABC（ABCモードへ）

Card Mode

⇄ CARDキーで切り替わります。カードのランクとスートを入力するためのモードです。

キー種別キー
ランクA 2〜10 J Q K
スート♠(s) ♣(c) ♥(h) ♦(d)
ナビゲーション・編集← → ␣ ⌫ ↵
モード切替ACT（アクションモードに戻る）

ABC Mode

ABCキーで切り替わります。OS標準のソフトキーボードが表示され、日本語入力やプレイヤー名などの自由テキストを入力できます。

ヒント: プレイヤー名を日本語で入力したい場合はABCモードに切り替えてください。入力が終わったらActionモードに戻すことで、再びポーカー専用キーボードが使えます。

19.オートスペース挿入

Actionモードで数字を入力した後にアクションキー（c r b f x ai win chop）を押すと、数字とアクションの間にスペースが自動挿入されます。

例

10 → c と押すと → 10 c と入力されます。金額が続く場合は 10 → c → 2000 で 10 c2000 となります。

ヒント: シート番号の後にいちいちスペースキーを押す必要がなくなり、入力スピードが大幅に向上します。

20.フルスクリーンエディタ

「テンプレート生成」ボタンを押すと、画面全体を使ったフルスクリーンのテキストエディタが開きます。カスタムキーボードと組み合わせて、ハンドのアクションを快適に記録できます。

  • テキストエリアが画面全体に広がり、長いハンド記録でも見やすい
  • カスタムキーボードが画面下部に固定表示される
  • テキストエリア下部のドラッグハンドルで高さを調整可能
  • 編集完了後は「Save」でセッションに保存、「Cancel」で破棄

21.カードピッカー

Hero CardsやBoard Cardsのスロットをタップすると、カードピッカーが表示されます。

操作方法

  1. カードスロットをタップして選択（選択中のスロットはハイライト表示）
  2. ランク（A, 2〜9, T, J, Q, K）をタップ
  3. スート（♠ ♣ ♥ ♦）をタップ
  4. カードが自動的にスロットに反映される
  • スロットは任意の順序で入力可能（左から順番に入力する必要なし）
  • 既に使用されたカードは重複して選択できない
  • 入力済みスロットの「×」ボタンでカードを除去可能

カラーカスタマイズ

22.カラーカスタマイズ

フッターのカラーパレットアイコンをタップすると、フルスクリーンのカラーカスタマイズ画面が開きます。アプリ全体の配色を自由に変更できます。

画面の構成

  • 上半分: ライブプレビュー — Journey・セッション・ハンドのサンプルが表示され、色の変更がリアルタイムで反映される
  • 下半分: カラーピッカー — 3つのカラー設定が表示される

3つのカラー設定

カラー用途デフォルト
基調色 (STANDARD)テキスト・ボーダー・アクセントカラー#144d75
中間色 (FILL)ヘッダー・ボタン・強調背景。透明度スライダー（5%〜100%）で濃さを調整可能#063b63
背景色 (BG)ページ全体の背景色#eff6f8

お気に入り（FAV）

  • 5つのお気に入りプリセットスロットが用意されています（初期状態で4パターンがプリセット済み）
  • 現在の配色をスロットに保存可能
  • スロットをタップで配色を読み込み
  • スロットの × ボタンで削除
  • 使用中のスロットに上書き保存する場合は確認ダイアログが表示されます

操作ボタン

  • Apply — 選択した配色を適用し保存（リロード後も維持）
  • Reset — デフォルトカラーに戻す
  • Close — 変更を適用せずに閉じる

24.PWA（Progressive Web App）インストール

このアプリはPWA対応です。PWA（Progressive Web App）とは、ウェブサイトをスマホやPCにアプリのようにインストールできる仕組みです。

インストールのメリット

  • ホーム画面からワンタップで起動（URLを開く手間なし）
  • ブラウザのアドレスバーやタブが非表示になり、画面を広く使える
  • 電波の届かない場所でもオフラインで利用可能
  • アプリストアからのダウンロードは不要で、容量もほとんど使いません

インストール方法

  • iOS: Safariで開く → 共有ボタン → 「ホーム画面に追加」
  • Android: Chromeで開く → メニュー → 「アプリをインストール」
  • PC: Chrome/Edgeのアドレスバー右のインストールアイコン

安全性について

PWAはGoogle、Microsoft、Appleが推進するウェブ標準技術です。個人情報の収集や外部サーバーへのデータ送信は一切行いません。

  • 本アプリのデータはすべてお使いの端末内（ブラウザのlocalStorage）にのみ保存されます
  • アカウント登録やログインは不要です
  • インターネット接続がなくても動作するため、データが外部に送られることはありません
  • PWA自体はブラウザのブックマークに近い仕組みで、端末の連絡先・写真・位置情報などへのアクセス権限を持ちません
