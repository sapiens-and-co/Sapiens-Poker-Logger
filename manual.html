Sapiens Poker Logger v0.2 - Manual

基本操作

1.概要

Sapiens Poker Loggerは、ポーカーの遠征記録・セッション管理・ハンド履歴をテキスト形式で記録するPWAアプリです。ブラウザだけで動作し、オフラインでも使えます。

データ保存: 全データはブラウザのlocalStorageに保存されます。アカウント登録は不要です。ブラウザのデータ削除や端末の故障で記録が失われる場合があるため、CSV ExportやHands Exportで定期的にバックアップを取ることをおすすめします。

2.トップページ（記録一覧）

アプリ起動時に表示される画面です。作成した記録（Trip）が一覧で並びます。

画面の構成

  • 「新しい記録を作成」ボタン — 新しい遠征・旅の記録を作成
  • 記録カード — 各記録のタイトル・日付・SESSIONS数・P/L・HOURS を表示。タップで中に入る
  • DELボタン — 各記録の右下にあり、記録を削除（確認ダイアログ付き）
  • App Guide — アプリの基本的な使い方の簡易ガイド

フッター

  • マニュアル — このマニュアルを開閉（どの画面からでも利用可能）
  • 要望を投稿 — フィードバックフォームを開く
  • SNSアイコン — X / YouTube / note へのリンク

3.記録（Trip）管理

旅行・遠征単位でセッションをグルーピングできます。

記録の作成

  1. トップ画面の「新しい記録を作成」をタップ
  2. タイトルを入力（例: 2025年11月 台湾ポーカー旅）
  3. 作成された記録をタップして中に入る

記録内の機能

  • 記録単位での累計統計（Total P/L / Sessions / Total Hours / BB/hr）
  • 「CSV Export」で収支データを出力（記録内に完了セッションがある場合に表示）
  • 「← 記録一覧」で トップ画面に戻る

セッション

4.セッション管理

セッション開始

  1. 記録内で「New Session」をタップ
  2. Location（場所）、Game（NLH/PLO）、SB/BB、Buy-in を入力
  3. 「Start」でセッション開始。経過時間の計測が始まります

セッション中の操作

ボタン機能
Add OnセッションバーのBuy-in表示横のボタン。追加バイインを記録。複数回可能
Endキャッシュアウト額を入力してセッション終了。収支が自動計算されます
Hands保存済みハンドの一覧表示。Export (.txt) で一括出力

セッションバー

セッション中は画面上部にセッション情報バーが表示されます。場所・ブラインド・経過時間・Buy-in合計が確認できます。

BB/hr計算: ブラインドが途中で変わった場合、各フェーズの経過時間で加重平均BBを算出し、正確なBB/hrを計算します。

5.セッション詳細の編集

終了済みセッションをタップすると詳細画面が開きます。各項目をタップして直接編集できます。

編集可能な項目

項目操作
Locationタップ → テキスト入力に切替。入力後に自動保存
Dateタップ → 日時編集モーダルが開く。開始・終了の日付と時刻を変更可能
Gameタップ → NLH / PLO のセレクトに切替
Buy-insタップ → 各Buy-in金額を個別に編集できるモーダルが開く
Cash-outタップ → 数値入力に切替

P/L・Duration・BB/hr・Hands は自動計算のため直接編集はできません。編集可能な項目には破線アンダーラインが表示されます。

セッションを再開

詳細画面の「セッションを再開」ボタンで、終了したセッションをアクティブ状態に戻せます。経過時間の計測も再開され、ハンドの追加記録やAdd Onが可能になります。

ヒント: キャッシュアウト額の入力ミスや、ハンドの追記忘れがあった場合に便利です。

  2. Cards でHeroカード・ボードカードを選択
  3. Players でプレイヤーの着席状況を設定
  4. 「テンプレート生成」→ アクションを追記 → Save

カスタムキーボード & 入力支援

17.カスタムキーボード NEW

ハンド記録の編集画面では、ポーカー専用のカスタムキーボードが表示されます。3つのモードを切り替えて効率的に入力できます。

Action Mode（アクションモード）

デフォルトのモードです。ハンドのアクションを素早く入力するためのキーが並びます。

キー種別キー
数字0〜9
記号# /
アクションx(check) c(call) b(bet) r(raise) f(fold) ai(allin) win chop
ナビゲーション← →（カーソル移動）
編集␣(スペース) ⌫(バックスペース) ↵(改行)
モード切替CARD（カードモードへ） ABC（ABCモードへ）

Card Mode（カードモード）

CARDキーで切り替わります。カードのランクとスートを入力するためのモードです。

キー種別キー
ランクA 2〜10 J Q K
スート♠(s) ♣(c) ♥(h) ♦(d)
ナビゲーション・編集← → ␣ ⌫ ↵
モード切替ACT（アクションモードに戻る）

ABC Mode（ABCモード）

ABCキーで切り替わります。OS標準のソフトキーボードが表示され、日本語入力やプレイヤー名などの自由テキストを入力できます。

ヒント: プレイヤー名を日本語で入力したい場合はABCモードに切り替えてください。入力が終わったらACTIONモードに戻すことで、再びポーカー専用キーボードが使えます。

18.オートスペース挿入 NEW

アクションモードで数字を入力した後にアクションキー（c r b f x ai win chop）を押すと、数字とアクションの間にスペースが自動挿入されます。

例

10 → c と押すと → 10 c と入力されます。金額が続く場合は 10 → c → 2000 で 10 c2000 となります。

ヒント: シート番号の後にいちいちスペースキーを押す必要がなくなり、入力スピードが大幅に向上します。

19.フルスクリーンエディタ NEW

「テンプレート生成」ボタンを押すと、画面全体を使ったフルスクリーンのテキストエディタが開きます。カスタムキーボードと組み合わせて、ハンドのアクションを快適に記録できます。

  • テキストエリアが画面全体に広がり、長いハンド記録でも見やすい
  • カスタムキーボードが画面下部に固定表示される
  • 編集完了後は「Save」でセッションに保存

20.リサイズハンドル NEW

テキストエリアの下部にドラッグハンドルが表示されます。モバイル端末でこのハンドルをドラッグすることで、テキストエリアの高さを自由に調整できます。

ヒント: ハンド記録が長くなった場合に、テキストエリアを広げて全体を見渡せるようにすると便利です。

21.カードピッカーキーボード NEW

ヒーローカード（Hero Cards）やコミュニティボード（Flop / Turn / River）のカード入力欄では、専用のカードピッカーキーボードが表示されます。カスタムキーボードと同じスタイルの統一されたUIで、ランクとスートをタップしてカードを入力できます。

操作方法

  1. カード入力スロットをタップして選択
  2. カードピッカーでランク（A, 2〜10, J, Q, K）を選択
  3. スート（♠ ♣ ♥ ♦）を選択
  4. カードが自動的にスロットに反映される

22.カードスロットのハイライト NEW

カード入力時、現在選択中のカードスロットが黄色の枠線とグロー効果でハイライト表示されます。これにより、どのスロットにカードが入力されるかが一目で分かります。

23.任意順序のカード入力 NEW

ヒーローカードとコミュニティボードのカードスロットは、任意の順序で入力できます。左から順番に入力する必要はなく、好きなスロットをタップして先に入力することが可能です。

例: フロップの3枚目だけ先に入力したり、ヒーローカードの2枚目から入力したりできます。

25.PWA（Progressive Web App）インストール

このアプリはPWA対応です。PWA（Progressive Web App）とは、ウェブサイトをスマホやPCにアプリのようにインストールできる仕組みです。

インストールのメリット

  • ホーム画面からワンタップで起動（URLを開く手間なし）
  • ブラウザのアドレスバーやタブが非表示になり、画面を広く使える
  • 電波の届かない場所でもオフラインで利用可能
  • アプリストアからのダウンロードは不要で、容量もほとんど使いません

インストール方法

  • iOS: Safariで開く → 共有ボタン → 「ホーム画面に追加」
  • Android: Chromeで開く → メニュー → 「アプリをインストール」
  • PC: Chrome/Edgeのアドレスバー右のインストールアイコン

安全性について

PWAはGoogle、Microsoft、Appleが推進するウェブ標準技術です。個人情報の収集や外部サーバーへのデータ送信は一切行いません。

  • 本アプリのデータはすべてお使いの端末内（ブラウザのlocalStorage）にのみ保存されます
  • アカウント登録やログインは不要です
  • インターネット接続がなくても動作するため、データが外部に送られることはありません
  • PWA自体はブラウザのブックマークに近い仕組みで、端末の連絡先・写真・位置情報などへのアクセス権限を持ちません
PWAの採用事例: X（旧Twitter）、Spotify、Starbucks、Pinterest、Uber など、世界的な大手サービスでもPWA技術が採用されています。安心してご利用ください。
